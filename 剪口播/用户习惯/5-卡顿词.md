# 卡顿词

## 模式

同一个词连续说 2-3 次：

```javascript
const stutterPatterns = [
  '那个那个',
  '就是就是',
  '然后然后',
  '这个这个',
  '所以所以'
];
```

## 🚨 英文 / 部分词卡壳（易漏，必查）

**检测不要只盯中文。** 英文术语/产品名最常见的是「**词头卡壳**」——先吐出词的开头一截，再说完整词：

```
ben benchmark   → 删「ben」，保「benchmark」
Mind Mindverse  → 删「Mind」，保「Mindverse」
ada adapter     → 删「ada」，保「adapter」
模式：(词的前缀) (空格/停顿)? (以该前缀开头的完整词)
```

判定：前一个英文片段是后一个完整英文词的**开头子串** → 是卡壳，删前缀。
（2026-06-17 漏过一次「ben benchmark」：中文导向规则没匹配，且核验脚本误把"纯英文重复"排除掉了。检测器与核验都要覆盖英文。）

## 删除策略

删前面，保留最后一个。

```
原文: "那个那个我想说"
删除: "那个"
保留: "那个我想说"
```
