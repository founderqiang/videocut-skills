## AI 审核顺序（按优先级）

1. **静音 >1s** → 建议删
2. **残句** → 删整句（话说一半 + 静音）
3. **重复** → 删前面，保后面（句间/句内/纠正，见4-重复检测.md）
4. **卡顿词** → 删前面（那个那个、就是就是）
5. **语气词** → 标记待人工确认（嗯啊呃）
6. **有口播稿** → 句子级对齐补漏，抓整句级口误（见11-口播稿对齐.md）

## 两条核心方法论

- **删前保后**（方向）：后说的通常更完整，删前面保后面。见 1-核心原则.md。
- **删除风险分层**（取舍）：删"字"安全、删"意思"危险；复核（verify）只砸高风险项，低风险交网页人工。见 10-删除风险分层.md。
